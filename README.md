# Conta Bancária em Java

Projeto simples em Java para praticar conceitos básicos de Programação Orientada a Objetos.

## Conceitos abordados
- Classes e objetos
- Encapsulamento (`private` e getters)
- Construtor
- `ArrayList`
- Laço `for-each`
- Estrutura condicional `if`

## Descrição
O programa cria objetos da classe `ContaBancaria`, armazena essas contas em um `ArrayList` e percorre a lista para comparar os saldos.  
Ao final, é exibida a conta que possui o maior saldo.

Projeto voltado para estudo e aprendizado de Java para iniciantes.
